✈️ Travel Booking App UI

## 📌 Project Overview

This is a *Travel Booking App UI Design* created using *Figma*. The application provides a simple and attractive interface for users to book travel destinations. It includes destination selection, booking details, payment, and booking confirmation screens.

---

## 🎯 Objective

The main objective of this project is to design a user-friendly travel booking application with a clean and modern interface that makes booking easy and convenient.

---

## ✨ Features

- Splash Screen
- Destination Selection
- Booking Details Form
- Booking Summary
- Price Breakdown
- Payment Method Selection
- Booking Confirmation Screen
- Clean and Modern UI Design
